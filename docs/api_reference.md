# API Reference

## Autenticación

### POST /auth/login
Inicia sesión y devuelve un token JWT.

### POST /auth/register
Registra un nuevo usuario.

## Pacientes

### GET /pacientes
Lista todos los pacientes registrados.

### GET /pacientes/{id}
Obtiene los datos de un paciente por su ID.

## Escalas

### GET /escalas
Lista las escalas disponibles.

### GET /escalas/{id}
Obtiene los resultados de una evaluación por su ID.
